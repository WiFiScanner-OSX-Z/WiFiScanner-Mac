## Overview

Network security assessment is a practical use case for <a href="#">WiFi Scanner</a> that goes beyond performance troubleshooting to include the verification that wireless networks in the vicinity are correctly secured. The security type display for each detected network — open, WEP, WPA, WPA2, WPA3 — provides an immediate overview of the security posture of nearby wireless infrastructure. For network administrators responsible for a facility's wireless security, the ability to verify that no unauthorized open access points have been deployed within range is a useful monitoring capability. For home users, the visibility into neighboring networks provides context for understanding the wireless environment.

The network detail information that <a href="#">WiFi Scanner</a> provides — including BSSID, vendor identification from the MAC address prefix, PHY mode, channel width, and current transmit rate for the connected network — is valuable for troubleshooting connection quality issues that involve the specific capabilities negotiated between the client device and the access point. Understanding whether a connection is operating in 802.11n or 802.11ac/Wi-Fi 5 mode, whether the full channel width is being utilized, and what transmit rate has been negotiated can reveal capability mismatches between client and access point that explain performance shortfalls below the theoretical maximum. The vendor identification from <a href="#">MAC address lookup</a> helps identify the manufacturer of each access point, which can be useful for matching detected networks to known hardware in an environment.

The export and logging capabilities in <a href="#">WiFi Scanner</a> support documentation workflows for network installers and administrators who need to produce records of the wireless environment before and after changes, verify that installed networks meet coverage requirements, or provide evidence for regulatory or compliance purposes. The ability to export scan results to CSV enables further analysis in spreadsheet tools, and the configurable scan history logging maintains an ongoing record of the wireless environment that documents changes over time. For any user who wants better understanding and control of their wireless environment, <a href="#">WiFiScanner OSX</a> provides the analytical tools that make wireless network management informed and effective.

## Key Features

- Visual <a href="#">channel graph</a> showing 2.4 GHz and 5 GHz spectrum occupancy at a glance
- Connected network details revealing <a href="#">transmit rate, PHY mode</a>, and link quality
- Vendor identification matching <a href="#">MAC address prefixes</a> to hardware manufacturers
- Network <a href="#">watchlist alerts</a> notifying when specific SSIDs appear or disappear
- Multiple adapter support scanning via <a href="#">built-in and external</a> wireless interfaces
- Ping and <a href="#">connectivity test</a> tools for verifying network access from scan results
- Sort and filter networks by <a href="#">signal, channel, security</a>, or vendor
- Location-aware scanning logging <a href="#">scan results with GPS coordinates</a> on supported hardware
- Hidden network detection revealing <a href="#">non-broadcasting SSIDs</a> in the vicinity
- Automatic <a href="#">scan history logging</a> for comparing network environments over time
